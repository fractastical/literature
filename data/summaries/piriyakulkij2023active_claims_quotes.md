## Key Claims and Hypotheses

1. **Main Claim:** The paper demonstrates that actively inferring user preferences by an LLM, guided by an entropy reduction algorithm, significantly outperforms baseline LLMs in terms of task performance while using fewer user interactions.

2. **Hypothesis:**  A probabilistic model, defined by prompting an LLM, can effectively guide an LLM to ask more informative questions, leading to faster and more accurate preference inference.

3. **Key Finding:**  The entropy reduction algorithm, by minimizing expected entropy and maximizing expected model change, allows the LLM to efficiently extract relevant information from user interactions, reducing the number of questions needed to achieve a desired level of preference understanding.

4. **Theoretical Contribution:** The paper proposes a framework for active preference inference using LLMs, combining probabilistic reasoning with a mechanism for selecting informative questions, representing a step towards more interactive and personalized AI systems.
