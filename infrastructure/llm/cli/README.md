# LLM CLI Module

Command-line interface for LLM operations.

## Quick Start

```bash
# Query the LLM
python3 -m infrastructure.llm.cli query "What is machine learning?"

# Check connection
python3 -m infrastructure.llm.cli check

# List models
python3 -m infrastructure.llm.cli models
```

## Commands

- `query` - Send queries to LLM
- `check` - Check Ollama connection
- `models` - List available models
- `template` - Apply research templates
